# usage
1. clone this repo
```commandline
git clone https://github.com/IAKSH/ImageSpider.git
```
2. install requirements
```commandline
pip install -r .\requirements.txt
```
3. launch `spider.py`
```commandline
python .\spider.py --keyword your_keyword --start_page 0 --end_page 5 --delete_duplicates=true
```
